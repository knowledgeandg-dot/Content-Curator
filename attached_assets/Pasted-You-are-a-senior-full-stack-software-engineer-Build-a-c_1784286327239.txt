Application Name:

DREAM VALLEY

====================================================

PROJECT PURPOSE

====================================================

Create an internal real estate inventory management system.

The company sells plots.

The purpose of this application is:

1. CRM Team manages and controls all inventory.
2. Sales Team only views currently available plots.
3. Sales Team does not contact CRM repeatedly for availability.
4. Inventory updates happen live.


There are only two interfaces:

1. CRM Dashboard
2. Sales Dashboard

====================================================

- Export Excel
- View Activity Logs


====================================================
ROLE 2: SALES USER
====================================================


Sales has view-only authority.

Sales cannot:

- Add inventory
- Edit inventory
- Delete inventory
- Change status
- Access CRM dashboard


Sales can only:

- Login using RM Code
- View available plots
- Search inventory
- Filter inventory
